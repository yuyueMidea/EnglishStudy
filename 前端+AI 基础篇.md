**前端+AI 基础篇：你最该吃透的 10 个点**
- 流式输出（SSE/streaming）+ 前端逐字渲染
- 上下文管理（截断/摘要/记忆）
- 工具调用（function calling）+ 参数校验
- 实时信息获取（搜索/浏览/爬取的产品化方式）
- 幻觉治理（拒答、引用、约束输出）
- 提示词工程（模板化、迭代、评估）
- RAG 全流程（Loader→切割→Embedding→检索→生成→引用）
- Agent 基础（多步任务、工具链、预算）
- 本地/多模型策略（成本/延迟/可用性）
- 最小上线能力（评估集+回退机制）

LangChain 基础篇：你最该吃透的 8 个点
- 模型接口 + 流式
- Prompt 模板化（结构化提示词）
- LCEL / Runnable（把链路工程化）
- 输出解析器（JSON/结构化/容错）
- Loader（本地/Web）
- 切割 + Embedding + 向量库
- 检索优化（过滤、rerank、topK）
- Memory + Tools + Agent（组合成系统）

平台篇（Coze/Dify）：你最该吃透的 6 个点
- 插件（把已有服务接成工具）
- 工作流（比 Agent 更稳定的编排）
- 条件选择/循环/批处理（自动化任务必备）
- 知识库/数据库（RAG 与结构化数据并存）
- 消息卡片/对话流（用户体验）
- 本地部署（可控性/合规）

最后一个“学习顺序建议”（不绕路）
- 流式输出 + 上下文（先把体验做出来）
- Function Calling + 结构化输出（让它能干活）
- RAG 实战 + 引用（让它有业务知识）
- 幻觉治理 + 评估 + 回退（让它能上线）
- LangChain 工程化（让系统可维护）

平台工作流（让你交付更快）

Agent / MCP / 多模型 / 本地部署（锦上添花或特定场景必需）
